Customer Requirement :

From contact entity  create an email using "template" It should display "FILTERED TEMPLATES" based on "contact contract"

Our Solution
In this approach, the flow will be;
1.	We have one ribbon button on Contact form called “Sent Contract Email”.
2.	User click on this button, one HTML page will open which contact one Grid table [for all the global Email template which contain specific subject word like “Cnt-”] and “open Email” button.
3.	User need to select any one of email template from list and click on “Open email” button.
4.	Once, user click on “Open email” button, Email form will be open which contain selected template’s content as subject and body of email.
5.	User can make modification to email record if needed and press Send button to send email to contact
